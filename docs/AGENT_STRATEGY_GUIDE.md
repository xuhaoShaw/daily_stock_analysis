# Agent 内置策略与行情对照指南

本文面向日常使用者，帮助你快速理解：

- 默认普通模式如何判断买卖
- Agent 策略模式如何启用
- 11 个内置策略分别适合什么行情
- 自动路由和手动点名策略时，应该怎么选

---

## 1. 先区分两种分析模式

### 默认普通模式

默认普通模式不会显式启用某个策略文件，而是走一套统一的趋势交易基线：

- 趋势优先：优先 `MA5 > MA10 > MA20`
- 不追高：价格偏离 `MA5` 过大时降低评级
- 偏好回踩：优先等待回踩 `MA5/MA10` 支撑
- 量价确认：缩量回调、放量突破优于无量上涨
- 风险排查：减持、处罚、诉讼、业绩变脸等利空会压低结论

它更像一个“默认多头趋势框架”，风格与内置 `bull_trend` 高度接近，但不要求用户显式选择策略。

### Agent 策略模式

Agent 策略模式会把内置 YAML 策略作为“分析视角”注入给模型，让模型按特定战法来观察同一只股票或 ETF。

适合以下场景：

- 你已经知道想看哪种方法，例如缠论、波浪、龙头
- 你想让系统在多种策略之间自动挑最匹配的几种
- 你希望输出里明确体现某个策略的触发条件、入场逻辑和止损位

---

## 2. Agent 策略模式怎么用

### Web / Bot / 对话入口

- Web 问股页：访问 `/chat`
- Bot 命令：`/ask`
- 自然语言点名：
  - `用缠论分析 600519`
  - `用放量突破分析 159915`
  - `比较一下龙头策略和情绪周期对某只题材股的看法`

### 常用配置方式

#### 方式一：单 Agent + 手动指定策略

适合你明确知道要用什么策略。

```env
AGENT_MODE=true
AGENT_ARCH=single
AGENT_SKILLS=chan_theory
AGENT_SKILL_ROUTING=manual
```

#### 方式二：多 Agent + 自动按行情选策略

适合不想手工切策略，希望系统按行情自动挑选。

```env
AGENT_MODE=true
AGENT_ARCH=multi
AGENT_SKILL_ROUTING=auto
```

#### 方式三：多 Agent + 固定策略列表

适合希望系统只在你关心的几种策略里做多视角交叉判断。

```env
AGENT_MODE=true
AGENT_ARCH=multi
AGENT_SKILL_ROUTING=manual
AGENT_SKILLS=ma_golden_cross,volume_breakout,shrink_pullback
```

### 关键配置项

| 配置项 | 作用 | 建议 |
|---|---|---|
| `AGENT_MODE` | 是否显式启用 Agent 策略问股 | 新手可先设 `true` |
| `AGENT_ARCH` | `single` 单 Agent；`multi` 多 Agent 编排 | 想看多策略共识时用 `multi` |
| `AGENT_SKILLS` | 手动指定启用的策略 id，逗号分隔 | 留空时默认回退到 `bull_trend` |
| `AGENT_SKILL_ROUTING` | `auto` 自动路由；`manual` 仅使用 `AGENT_SKILLS` | 常规建议 `auto` |
| `AGENT_LITELLM_MODEL` | Agent 专用模型 | 可留空继承主模型 |
| `AGENT_MAX_STEPS` | Agent 最大步数 | 不熟悉时保持默认 |

## 3. 自动路由怎么看“当前行情”

当 `AGENT_ARCH=multi` 且 `AGENT_SKILL_ROUTING=auto` 时，系统会先判断当前更像哪类行情，再从匹配的策略里选择最多 3 个。

### 行情标签

| 行情标签 | 含义 | 典型特征 |
|---|---|---|
| `trending_up` | 趋势上行 | 均线多头、趋势分高 |
| `trending_down` | 趋势下行 | 均线空头、趋势分低 |
| `sideways` | 横盘震荡 | 趋势不明、分数居中 |
| `volatile` | 高波动结构市 | 波动加大、量能活跃但方向未定 |
| `sector_hot` | 题材或板块热度市 | 板块轮动明显、热点集中 |

### 自动路由的一般规则

1. 用户在提问里明确点名的策略，优先级最高
2. 若使用 `manual`，只看 `AGENT_SKILLS`
3. 若使用 `auto`，按当前行情标签匹配 `market_regimes`
4. 若没有明显匹配，回退到默认路由策略

### 默认回退逻辑

- 默认激活主策略：`bull_trend`
- 默认路由回退策略：`bull_trend`、`shrink_pullback`

也就是说，就算你没有手动指定策略，系统通常也会优先落回“趋势延续 + 回踩低吸”这一类风格。

## 4. 11 个内置策略与适用行情总表

| 策略 id | 显示名称 | 类别 | 更适合的行情 | 更适合的标的 | 核心关注点 |
|---|---|---|---|---|---|
| `bull_trend` | 默认多头趋势 | `trend` | 上升趋势 | 普通股票、ETF | 多头排列、回踩不破、不追高 |
| `ma_golden_cross` | 均线金叉 | `trend` | 盘整转强、上升初段 | 普通股票、ETF | MA 金叉 + 量能确认 |
| `volume_breakout` | 放量突破 | `trend` | 平台突破、前高突破 | 强势股、ETF | 阻力位上方放量站稳 |
| `shrink_pullback` | 缩量回踩 | `trend` | 趋势中继回调 | 趋势股、ETF | 缩量回踩 MA5/MA10 后企稳 |
| `bottom_volume` | 底部放量 | `reversal` | 长跌后反转尝试 | 超跌股 | 底部异动、阳线企稳、严格止损 |
| `box_oscillation` | 箱体震荡 | `framework` | 横盘震荡 | 区间股、宽幅 ETF | 箱底买、箱顶卖、假突破识别 |
| `emotion_cycle` | 情绪周期 | `framework` | 高热度、题材轮动 | 热点题材股 | 情绪低点布局、情绪高点减仓 |
| `dragon_head` | 龙头策略 | `trend` | 板块启动、行业催化 | 题材龙头股 | 板块强度、相对强势、换手与催化 |
| `chan_theory` | 缠论 | `framework` | 结构复杂、高波动 | 波段股、结构派 | 中枢、背驰、一买二买三买 |
| `wave_theory` | 波浪理论 | `framework` | 波段推进或调整明显 | 波动较大的趋势股、ETF | 浪型位置、斐波那契支撑与目标 |
| `one_yang_three_yin` | 一阳夹三阴 | `pattern` | 强势整理后续涨 | 强势股 | 五日 K 线形态、整理后再启动 |

## 5. 每个内置策略分别适合什么行情

### 5.1 `bull_trend` 默认多头趋势

**适合：**

- 常规个股分析
- 明确上升趋势
- 你不想追热点，只想做顺势低风险买点

**核心信号：**

- `MA5 >= MA10 >= MA20`
- 趋势向上但价格没有严重偏离均线
- 回踩关键均线后企稳
- 放量突破或健康量价配合

**不适合：**

- 纯箱体震荡
- 明显空头下跌
- 题材情绪极端波动阶段

### 5.2 `ma_golden_cross` 均线金叉

**适合：**

- 盘整结束准备转强
- 上升趋势的启动初段
- 想找“趋势刚拐头”的用户

**核心信号：**

- `MA5` 近 3 个交易日上穿 `MA10`
- 更强的是 `MA10` 上穿 `MA20`
- 金叉当天量能放大
- 最好还有 MACD 金叉共振

**不适合：**

- 深度下跌后的弱反弹
- 已经涨了一大段、严重偏离均线的位置

### 5.3 `volume_breakout` 放量突破

**适合：**

- 平台突破
- 接近前高或箱体上沿
- 强势股二次加速

**核心信号：**

- 突破 20 日高点或平台顶部
- 成交量显著高于均量
- 收盘站稳阻力位上方
- 突破后乖离率仍可接受

**不适合：**

- 低量假突破
- 突破后离均线过远
- 板块没有共振、单兵突进

### 5.4 `shrink_pullback` 缩量回踩

**适合：**

- 主升趋势中的正常回调
- 趋势股的二次低吸
- ETF 或宽基中短线回踩确认

**核心信号：**

- 前提仍是上升趋势
- 回踩 `MA5` 或 `MA10`
- 回调过程明显缩量
- 当前价格守住均线支撑

**不适合：**

- 破位下跌
- 回踩过程中放量杀跌
- 本来就不是多头结构的股票

### 5.5 `bottom_volume` 底部放量

**适合：**

- 长期下跌后出现反转试探
- 超跌股首次出现大成交异动

**核心信号：**

- 20 日高点到近期低点跌幅较大
- 趋势状态偏空
- 突然放量，但当日收阳并守住低点
- 可能伴随新闻催化或成本收敛

**不适合：**

- 普通趋势跟随用户
- 没有任何企稳迹象、只是加速下跌

**风险提示：**

- 这类策略天然比趋势跟随更激进
- 仓位通常应更轻，止损更严格

### 5.6 `box_oscillation` 箱体震荡

**适合：**

- 横盘整理
- 价格长期在固定区间来回波动
- 想做区间波段而不是趋势追涨

**核心信号：**

- 箱顶与箱底反复被验证
- 当前价格靠近箱底时更有性价比
- 靠近箱顶时偏减仓或观望
- 放量突破则要切换为趋势思路

**不适合：**

- 单边趋势主升或主跌阶段
- 箱体宽度太小、没有波动空间

### 5.7 `emotion_cycle` 情绪周期

**适合：**

- 高热度题材股
- 短线情绪驱动明显的市场阶段
- 你想判断“情绪底/情绪顶”

**核心信号：**

- 换手率、成交量、新闻热度的阶段变化
- 恐慌阶段偏低吸
- 狂热阶段偏减仓
- 高乖离率和顶背离常是危险信号

**不适合：**

- 长期价值持有分析
- 低换手、低弹性的慢牛标的

### 5.8 `dragon_head` 龙头策略

**适合：**

- 板块启动
- 题材扩散
- 行业或政策催化落地

**核心信号：**

- 所在板块处于强势轮动阶段
- 个股跑赢板块平均表现
- 换手率和量比活跃
- 有明确板块级催化剂

**不适合：**

- 普通跟风票
- 没有板块共振、只是单股异动
- 已经过热且乖离过大的尾声位置

### 5.9 `chan_theory` 缠论

**适合：**

- 结构复杂的波段行情
- 你更关注中枢、背驰和级别买点
- 想要比普通趋势模式更强调结构位置

**核心信号：**

- 是否处于中枢震荡还是趋势段
- 有无顶背驰或底背驰
- 当前属于一买、二买、三买还是暂无明确买点

**不适合：**

- 想快速得到极简买卖建议的新手
- 低波动、结构不明显的标的

### 5.10 `wave_theory` 波浪理论

**适合：**

- 推动浪和调整浪较明显的品种
- 想看当前更像第几浪
- 想结合斐波那契位置判断目标与回调

**核心信号：**

- 当前浪型位置
- 第 2 浪或第 4 浪回调企稳
- 第 3 浪突破确认
- 第 5 浪末端避免追高

**不适合：**

- 想要完全客观、无主观归数的分析方式
- 波动太小、浪型不明显的标的

### 5.11 `one_yang_three_yin` 一阳夹三阴

**适合：**

- 强势股短整理
- 最近 5 日 K 线结构清晰
- 想做形态战法而不是大框架分析

**核心信号：**

- 先大阳，再三根缩量小阴或整理，再重新阳线突破
- 三根整理 K 线不破关键实体区域
- 若同时叠加多头排列，信号更强

**不适合：**

- 没有明显强势前导阳线
- 纯粹弱势下跌中的随机反抽

**使用提示：**

- 该策略更适合手动点名使用
- 自动路由不会优先选择它作为行情标签策略

## 6. 按常见场景怎么选策略

### 场景一：普通股票 / 宽基 ETF / 行业 ETF

优先顺序建议：

1. `bull_trend`
2. `shrink_pullback`
3. `ma_golden_cross`
4. `box_oscillation`
5. `wave_theory`

### 场景二：趋势股、突破股

优先顺序建议：

1. `volume_breakout`
2. `ma_golden_cross`
3. `bull_trend`
4. `shrink_pullback`

### 场景三：题材热点、短线龙头

优先顺序建议：

1. `dragon_head`
2. `emotion_cycle`
3. `volume_breakout`

### 场景四：超跌反转、抄底尝试

优先顺序建议：

1. `bottom_volume`
2. `chan_theory`
3. `wave_theory`

### 场景五：结构派、波段派

优先顺序建议：

1. `chan_theory`
2. `wave_theory`
3. `box_oscillation`

### 场景六：K 线形态玩家

优先顺序建议：

1. `one_yang_three_yin`
2. `ma_golden_cross`
3. `shrink_pullback`

## 7. 新手怎么选最稳妥

如果你不确定自己应该用哪个策略，可以直接用下面这套简化建议：

- **只想稳一点**：`bull_trend`
- **想找低吸点**：`shrink_pullback`
- **想找启动点**：`ma_golden_cross`
- **想追强势突破**：`volume_breakout`
- **想做题材龙头**：`dragon_head`
- **想看结构和级别**：`chan_theory`
- **想博超跌反转**：`bottom_volume`

如果你只看 ETF，建议先从以下几个开始：

- `bull_trend`
- `shrink_pullback`
- `ma_golden_cross`
- `box_oscillation`
- `wave_theory`

## 8. 使用建议与常见误区

### 建议

- 不熟悉策略时，先用单个策略，不要一开始就 `all`
- 先看“适用行情”是否匹配，再看结论
- 把策略输出当作“分析视角”，不是机械交易指令
- 题材股策略不宜直接套在低波动 ETF 上

### 常见误区

- **误区 1：把所有策略同时开起来就更准**
  - 不一定。策略越多，视角越杂，适合先做有针对性的筛选。

- **误区 2：反转策略能代替趋势策略**
  - 不能。`bottom_volume` 这类策略更激进，容错率更低。

- **误区 3：点名了策略就一定能得到强结论**
  - 不会。若当前行情不满足该策略触发条件，系统仍可能给出观望或等待。

- **误区 4：自动路由会覆盖一切主观选择**
  - 不会。你在问题里明确点名的策略优先级最高。
